# วิธีติดตั้ง

สามารถติดตั้งได้โดยใช้คำสั่ง

รุ่นเสถียร

```sh
$ pip3 install pythainlp
```
รุ่นกำลังพัฒนา

```sh
$ git clone https://github.com/wannaphongcom/pythainlp.git
$ cd pythainlp
$ python setup.py install
```
